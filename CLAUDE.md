## What this repository is

This is a **training/documentation repository**, not a software project. It contains course material (in Brazilian Portuguese) teaching Git and GitHub fundamentals to development teams. There is no application code, build system, dependency manifest, or test suite.

The substantive content lives entirely in `README.md`, which is a self-contained 29-section course (concepts, command reference, professional branch/PR workflow, commit conventions, and hands-on lab exercises). `LICENSE` is the only other tracked file.

## Working in this repo

- Edits almost always mean changing `README.md`. Preserve its existing structure: numbered `#` sections, `---` horizontal rules between topics, and fenced code blocks (` ```bash ` for commands, ` ```txt ` for diagrams/output).
- All prose is in Brazilian Portuguese. Match this language and the existing didactic, example-driven tone when adding or editing content.
- The README documents commit conventions used here too: follow Conventional Commits in Portuguese (e.g. `feat: adiciona login por Google`, `fix: corrige cálculo de comissão`, `docs: atualiza README`).
